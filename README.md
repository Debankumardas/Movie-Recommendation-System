
# 🎬 Movie Recommendation System

This project is a **Movie Recommendation System built using Python and Machine Learning techniques**.  
It recommends movies based on **content similarity, user ratings, and popularity**.

The system uses **Content-Based Filtering and Collaborative Filtering** to generate personalized movie recommendations.

---

## 🚀 Features

- 🔍 Search movies by **title, genre, or director**
- 🎯 **Content-based recommendations** using TF-IDF and Cosine Similarity
- 👥 **Collaborative filtering** using user ratings
- ⭐ **Personalized movie suggestions** for users
- 📊 Show **popular movies based on ratings**
- 🧠 Hybrid recommendation system
- 📈 Clean and structured dataset generation

---

## 🛠 Technologies Used

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**

- **TF-IDF Vectorizer**
- **Cosine Similarity**
